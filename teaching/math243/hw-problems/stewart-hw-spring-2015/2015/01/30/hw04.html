Solution to number 21

Let $f(x) = \int_0^x (1-t^2)e^{t^2} dt$. The first thing to notice is that $(1-t^2)e^{t^2}$ is a product of continuous functions on $[0,\infty )$. So we can immediately apply the fundamental theorem of calculus to get that $f’(x) = (1-x^2)e^{x^2}$. Since we are wanting to know when $f(x)$ is increasing, we need to find when $f’(x)$ is positive. To do this, we realize that $f’(x)$ determines the slope of tangent lines at each point in the domain of $f’(x)$. Thus the critical points (or the points where the slope is zero) partition the domain of $f’(x)$. To find these points, we set $f’(x) = 0$ and solve for $x$. Since $e^{x^2}>0$, we can ignore it and only consider $x$ when $1-x^2 = 0$. This happens when $x=-1,1$. These points partition the domain into three intervals, $(-\infty,-1)$, $(-1,1)$, and $(1,\infty)$. Testing values of $f’(x)$ in each of these intervals shows that $f’(x)>0$ when $-1<x<1$. Hence $f(x)$ is increasing on $(-1,1)$.
